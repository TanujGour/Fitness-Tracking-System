📌 Overview

The Fitness Training and Management System is a full-stack web application designed to help users track, manage, and improve their fitness journey.

It provides a seamless experience for monitoring daily activities, planning workouts, managing health data, and visualizing progress — all in one place.

✨ Features
🔐 Authentication
User Registration & Login
Google Login Integration
Secure JWT-based authentication

📊 Dashboard
Track daily steps and calories
Visual charts for progress
Quick overview of fitness activity

🏃 Activity Tracking
Step tracking
Calorie monitoring
Daily fitness insights

🥗 Planner & Meals
Workout planner
Meal planning system
Organized fitness schedules

❤️ Health Management
Water intake tracking
Sleep monitoring
BMI calculation

👤 Profile Management
View and edit profile details
Google profile photo integration
Upload custom profile image

🌙 UI/UX
Modern responsive design

Clean and user-friendly interface

⚙️ Installation (Local Setup)

1️⃣ Clone the repository
git clone https://github.com/TanujGour/Fitness-Tracking-System.git
cd Fitness-Tracking-System

2️⃣ Setup Backend
cd backend
npm install

Create .env file:
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key

Run backend:
npm start

3️⃣ Setup Frontend
cd frontend
npm install
npm start
